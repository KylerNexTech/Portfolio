Version History

A career and a life, read like a commit log. Ordered chronologically by v'decades'.'years'.'months'.

  1. v0.1.0Early direction

    First boot. Surrounded by electronic devices. Born curious, destined to learn.

  2. v0.4.0Turning point

    Fell in love with games, starting paying attention to their design and concepts.

  3. v0.8.0Journey begins

    Wrote my first line of code and never really stopped. But I wanted to make even bigger projects.

  4. v1.2.0Breaking change

    Starting creating Roblox games. Learned how to work with a team and create a product. Learned how to make a user focused product.

  5. v1.4.0Growth phase

    Started managing my own teams. Got faster and more deliberate at the same time. Turns out those aren't opposites.

  6. v1.6.0Current day

    Still learning and growing. Each project more refined than the last.
